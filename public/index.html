For AI agents

Connect the MCP server once, then just ask your agent to share a file:

claude mcp add --transport http --scope global transfer /mcp

The server exposes upload_file, list_files, update_file, delete_file, and get_usage tools, plus a usage-guide resource. Login happens in the browser on first connect.

Avoid upload permission prompts

Uploads PUT directly to this bucket, so some agent runtimes (e.g. Claude Code's auto mode) flag that request as a possible data-exfiltration pattern and block it the first time, even though the URL only ever points here.
